# Workout Engine

Нативное iOS-приложение для интервальных тренировок: создавайте свои пресеты с фазами «подготовка / работа / отдых», запускайте таймер с звуком и тактильной отдачей, отслеживайте прогресс по кругам.

## Возможности

### Тренировка
- **Быстрый старт** — последний использованный интервал или первый из сохранённых.
- **Список интервалов** — карточки с длительностью и количеством фаз; свайп для удаления.
- **Активная тренировка** — кольцевой прогресс фазы и всей сессии, пауза, пропуск фазы, остановка.
- **Фоновый таймер** — сессия продолжается при заблокированном экране за счёт аудиосессии, silent keep-alive и догона по wall-clock (см. [`docs/background-workout.md`](docs/background-workout.md)).

### Конструктор
- Название интервала, сводка по времени и визуальная шкала фаз в круге.
- Настройка **количества кругов** (1–99): один шаблон фаз повторяется на каждый круг.
- Фазы трёх типов: подготовка, работа, отдых — с длительностью и stepper’ом.
- Перетаскивание фаз за ручку, удаление свайпом (минимум одна фаза в круге).
- Сохранение пресетов в локальное хранилище.

### Настройки
- Тема: системная / светлая / тёмная.
- Звуки по типам фаз, отсчёт последних 3 секунд, общий выключатель.
- Тактильная отдача.
- Запрет гашения экрана во время тренировки.

### Локализация
- Русский и английский (`Localizable.xcstrings`); язык контента следует системной локали (ru → русский, иначе английский).

## Архитектура

```
workout-engine/
├── Engine/              # WorkoutEngine, развёртка циклов, последовательность фаз
├── Models/              # WorkoutPreset, PhaseKind, StoredPreset
├── Services/            # PresetStore, AppSettings, звук, хаптики, координатор сессии
├── Features/
│   ├── Home/            # список пресетов, быстрый старт
│   ├── Editor/          # конструктор интервалов
│   ├── Workout/         # экран активной тренировки
│   └── Settings/
└── Resources/           # звуки, строки локализации
```

**WorkoutEngine** — ядро таймера: загрузка пресета, переходы между фазами, пауза, обратный отсчёт, колбэки для звука и вибрации через `WorkoutFeedbackHandling`.

**PresetCycleExpander** разворачивает шаблон одного круга в полную последовательность фаз с учётом `roundCount`.

**PresetStore** читает и сохраняет пресеты через SwiftData; при запуске выполняет миграцию legacy-схемы и удаляет устаревшие встроенные пресеты.

## Требования

- macOS с **Xcode** (версия с SDK iOS 26.5)
- Симулятор или устройство на iOS 26.5+

## Сборка и запуск

1. Клонируйте репозиторий и откройте `workout-engine.xcodeproj` в Xcode.
2. Выберите схему **workout-engine** и целевое устройство.
3. **Product → Run** (⌘R).

## Тесты

```bash
xcodebuild test \
  -scheme workout-engine \
  -destination 'platform=iOS Simulator,name=iPhone 16'
```

Покрытие включает движок таймера, развёртку циклов, парсинг длительности, Quick Start и модель пресетов.
